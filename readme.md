# SW역량평가


## 프로젝트세팅

``` sh
$ cd ./server/
$ npm install -g nodemon
$ npm install -g babel-cli
$ npm install
$ npm run init
=> server/config/index.js 수정
$ npm run init
$ pip install -r .\python\requirements.txt
$ npm run init
```
## 서버실행
``` sh
$ npm run server
```
